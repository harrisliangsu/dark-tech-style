MATURITY MODEL
Where Tessera sits today.
A four-level scale of how much of the deployment loop is automated. Each level locks in something the previous one didn't.
L0
L0 Manual deploys
SSH and a tarball. Whoever shipped last knows how it works.
role: hero engineer
L1
L1 CI exists, CD doesn't
Tests run on PRs. Production still needs a human to click "Deploy".
role: gatekeeper
L2
L2 Continuous delivery
Every green commit reaches a staging environment automatically. Promotion to prod is one click.
role: reviewer
L3
L3 Hardened pipeline current
Six gates with hard exit codes. Canary, soak, automated rollback. Humans only intervene on failure.
role: pipeline architect
L4
L4 Self-healing target
Pipeline diagnoses its own failures, reroutes around bad nodes, opens its own incident reviews.
role: SRE strategist
